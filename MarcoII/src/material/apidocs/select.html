API

Directive: MdcSelectTextDirective

Selector: [mdcSelectedText]

Directive for the text representation of an mdcSelect selection control.

Directive: MdcSelectDirective

Selector: [mdcSelect]:not(select)

Directive for a spec aligned material design 'Select Control', build on top of a simple menu. This directive should wrap an MdcSelectTextDirective, and an MdcSimpleMenuDirective

Property Description
@Output()

valueChange

EventEmitter<any>

Event emitted when the value of the select changes. Note that when an mdcSelect is used as a FormControl, it's also possible to bind to ngModelChange instead of valueChange.
@Input()

value

any

Property for the chosen value of the select control. The value that each option represents can be set with the value option on the MdcListItemDirective that represents that choice. Note that when an mdcSelect is used as a FormControl, it's also possible to bind to ngModel instead of value.
@Input()

disabled

any

When this input is defined and does not have value false, the select control will be disabled.

Directive: MdcFormsSelectDirective

Selector: [mdcSelect][formControlName]:not(select) [mdcSelect][formControl]:not(select) [mdcSelect][ngModel]:not(select)

Directive for adding Angular Forms (ControlValueAccessor) behavior to an MdcSelectDirective. Allows the use of the Angular Forms API with select inputs, e.g. binding to [(ngModel)], form validation, etc.

Directive: MdcSelectNativeDirective

Selector: select[mdcSelect]:not([multiple])

Directive for a 'Select Control' based on the native HTML select element. This directive may provide better usability then the MdcSelectDirective on mobile devices. For tablets and desktop, the standard MdcSelectDirective is recommended. When this directive is used, the standard HTML option and optgroup elements must be used to define the choices. No additional directives are needed.

Directive: MdcSelectOptionNativeDirective

Selector: option[mdcSelectOption]

Directive for an option inside a mult-selection 'Select Control' using MdcSelectMultipleNativeDirective.

Directive: MdcSelectGroupNativeDirective

Selector: optgroup[mdcSelectGroup]

Directive for an optgroup inside a mult-selection 'Select Control' using MdcSelectMultipleNativeDirective.

Directive: MdcSelectMultipleNativeDirective

Selector: select[mdcSelect][multiple]

Directive for a mult-selection 'Select Control' based on the native HTML select[multiple] element. When this directive is used, the standard HTML option and optgroup elements must be used to define the choices. Each option should be annotated with an MdcSelectOptionNativeDirective, and each optgroup with an MdcSelectGroupNativeDirective. Option dividers can be created as follows: <option mdcListDivider></option>.
